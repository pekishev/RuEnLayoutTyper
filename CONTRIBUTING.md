# Contributing

Спасибо за интерес к проекту!

## Быстрый старт

Это Chrome‑расширение MV3 без сборки/зависимостей.

1) Откройте `chrome://extensions`
2) Включите **Developer mode**
3) **Load unpacked** → выберите папку проекта
4) После правок нажимайте **Reload**

## Как проверить изменения

- Откройте вкладку, куда хотите печатать.
- В попапе расширения вставьте тестовый текст и нажмите “Напечатать (CDP)”.
- Для web‑KVM рекомендуемый тест:
  - `<> , .`
  - `<!doctype html>`
  - смешанный текст: `hello привет world`

## Стиль и принципы

- Старайтесь сохранять логику ввода максимально “клавиатурной” (keydown/keyup + модификаторы).
- Не добавляйте тяжёлые зависимости.
- Изменения в маппинге RU→EN — в `mapping.js`.
- Изменения в CDP‑вводе — в `background.js`.

## Pull Requests

В PR желательно приложить:

- описание проблемы/цели
- пример входного текста
- где воспроизводится (обычный input / CodeMirror / web‑KVM и т.п.)
